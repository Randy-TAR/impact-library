## WHAT IS POSSIBLE NOW (Based on Your Endpoints)

### Strangers (No Login)
- View all books
- Search books by title/author/category
- View single book details
- Download books (PDF files)

### Logged-in Regular Users
- Same as strangers (no additional features)

### Librarians (Logged in)
- All stranger features
- Upload new books (with PDF)
- Update/edit book details
- Issue books to borrowers (create borrow record)
- View all borrow records
- Mark books as returned
- View dashboard statistics

### Admins (Logged in)
- All librarian features
- Delete books
- Register new users (create accounts)

---

## WHAT IS NOT POSSIBLE

- Users requesting books
- Users seeing their own borrowed books
- Users returning books themselves
- Online self-service borrowing
- Email notifications
- Late fee calculation
- Book reservation/waiting list
- User borrowing history
- Automatic due date reminders
- Public user registration (only admin can create accounts)
